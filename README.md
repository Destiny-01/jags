# jags
Javascript API Generator for Solidity Programs

Pending when the docs will be written
<b>Clone</b> into the environment
```
git clone https://github.com/galadd/jags.git
```
<b>Usage</b> 
```
node src/gen.js
```
<b>Install Dependencies</b> 
```
npm install
```
<b>Run Tests</b>
```
npm test
```
